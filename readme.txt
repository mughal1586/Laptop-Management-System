C++ Laptop Management System

Description
This C++ program manages laptop data. It defines a `laptop` class to store laptop details and provides functionality to insert and display laptop information.

Features
-   Defines a `laptop` class with attributes for brand, model, serial number, color, price, processor speed, RAM, and screen size.
-   Allows setting and getting laptop details.
-   Provides functions to insert laptop data and display laptop information.

Code Structure
The code consists of the following files:
-   `header.h`:  Contains the declaration of the `laptop` class.
-   `Laptops Managment System.cpp`: Contains the implementation of the `laptop` class methods.
-   `main.cpp`: Contains the main function that uses the `laptop` class to create laptop objects, set their data, and display their information.

Dependencies
-   iostream: For input/output operations.
-   string: For using string data type.

Compilation and Execution
To compile and run the program, save the files with their respective .cpp and .h extensions (e.g., header.h, Laptops Managment System.cpp, main.cpp) and use a C++ compiler.
